Proyecto de redes

el propósito de este proyecto es crear una aplicación que pueda escanear una red, este escaneo nos dará datos importantes de los equipos conectados y con estos datos poder tener una visión de la seguridad que hay en esos equipos, como por ejemplo puertos abiertos que hay en los equipos.
